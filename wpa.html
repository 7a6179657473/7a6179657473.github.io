WPA2 hash Gathering Project

Exploring Network Security and Wireless Authentication trends

Introduction

This project focuses on analyzing WPA2 protocols, their vulnerabilities, and the techniques used to gather relevant network data securely and ethically examine it.

Objective

  • Understand sqwuence of events during WPA2 authentication and encryption.
  • Collect a wide array of hashs valid for hashcat bruteforcing.
  • Crack & identify hashs.

Methodology

The project was carried out using ethical hacking techniques, including:

  • Network packet sniffing using software & hardware tools like Wireshark, Pineapple MK7, Pwnagotchi.
  • Analysis of WPA2 handshakes, discerning between half handshakes and full handshakes.
  • Attack the hash using Hashcat, gathering succesfull * un-succesfull attempts, specifically utalizing the rockyou wordlist & a combination of rockyou and other small wordlists.

Results

The study highlighted potential trends in end user WPA2 configurations and provided insights user behaivor.

Gathering the data involves various tools monitoring the raw traffic of WPA2 routers. The main focus is on the 2.4ghz frequency but on occasion 5ghz is targeted.

Conclusion

By understanding the behaivor and risks associated with password selection, this project aims to gain a more personal connection with popular trends on password security & the regions they are captured from.

During testing, an expected trend of personally named Wifi networks resulted in higher liklihoods of weak passwords.
